# The SwapX V3 SDK

> **Welcome to the V3 SwapX SDK!**

The SwapX V3 SDK provides abstractions to assist you with interacting with the SwapX V3 smart contracts in a Typescript/Javascript environment.
## Installation

To interact with the V3 SDK we recommend installing though npm:

```bash
npm i --save @swapx/v3-sdk
npm i --save @swapx/sdk-core
```
